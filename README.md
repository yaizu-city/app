# スマートマップ管理用リポジトリ

## 📘 テーマ図
#### ◾️ 操作ファイル
- theme/config.json
- theme/任意のファイル名.txt
#### ◾️ 操作手順
1. ダウンロードしたstyle.txtを任意のファイル名に変更し、themeフォルダに配置します。
2. 指定したファイル名を、`config.json`に記載します。
